# Font Size

The font-size CSS property sets the size of the font.

_Font size converted "px" to â†’ "rem"_

### For responsive

add `sm, md, lg, xl or xxl` with colon `:` before to class name for example:

```bash
<div class="fs12 md:fs16 xl:fs24"></div>
```
