# List of Content

for **responsive** add `sm, md, lg, xl or xxl` with colon `:` before to class name for example:

```bash
<div class="relative md:absolute xl:fixed"></div>
```

## position

```css
.relative {
  position: relative;
}

.absolute {
  position: absolute;
}

.fixed {
  position: fixed;
}

.static {
  position: static;
}

.sticky {
  position: sticky;
}
```
